# detecteur-de-port-du-casque
un modèle de Deep Learning basé sur une architecture de réseaux de neurones convolutifs (CNN). Le système est capable de classifier des images en deux catégories : "Sécurisé" (avec casque) ou "Danger" (sans casque).
Langage : Python

Frameworks IA : TensorFlow / Keras

Vision par Ordinateur : OpenCV

Interface Utilisateur : Gradio

Dataset : ~25 000 images traitées

Installation & Utilisation
Clonez le dépôt.

Installez les dépendances : pip install tensorflow opencv-python gradio.

Lancez le notebook pour entraîner ou charger le modèle .keras.

Utilisez l'interface Gradio pour tester vos propres images.
